# Reference

Technical descriptions of the Flaps API and data model. Reference material is
information-oriented: it describes the system as it is.

- **Admin REST API**: generated from the OpenAPI specification (available once
  the v0.1.0 server crate is published).
- **OFREP endpoint**: follows the [OFREP specification](https://github.com/open-feature/protocol).
- **Flag model**: see [Architecture](../design/architecture.md) for the domain
  model and [Interoperability](../design/interop.md) for the flagd ruleset format.
- **Configuration reference** (`flapsd` TOML): inline documentation in
  `flapsd::config` (generated by `cargo doc`).
